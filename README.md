# ai-interview-coach
AI Interview Coach is a JavaScript-based web application that simulates interview questions and evaluates user responses by providing feedback and scores. It also supports voice input, helping users practice and improve their communication skills.
##  Developed by PREMSANKAR PRADHAN

---
##  Overview
AI Interview Coach is a web application that helps users practice interview questions and receive instant feedback on their answers. It evaluates responses based on length and keywords to improve communication and confidence.

---

##  Features
-  Random interview questions
-  AI-like answer evaluation
-  Score system (out of 10)
-  Voice input using Speech Recognition
-  Modern and responsive UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Web Speech API

---

##  How to Run
1. Download or clone the repository
2. Open `index.html` in your browser

---

## 💡 Use Case
This project helps students and job seekers improve their interview performance by practicing answers and receiving feedback.
